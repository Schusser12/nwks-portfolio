Vještine i rad

Pregled područja u kojima imam najviše iskustva i interesa.

Tehničke vještine

Najviše iskustva imam s Linux sustavima, Bash skriptama, analizom logova i osnovnim web tehnologijama. Ovaj portfolio je napravljen pomoću HTML-a, CSS-a i malo JavaScripta, tako da se kroz projekt vidi osnovna struktura statičkog web mjesta.

Linux Bash Shell scripting Log analysis HTML CSS JavaScript UX osnove

Dio javnih radova i skripti može se pronaći na GitHub profilu: github.com/Schusser12

Bash skripte

Bash skripte koristim za automatizaciju ponavljajućih zadataka, brže provjere sustava, izdvajanje korisnih informacija iz logova i jednostavnije snalaženje u terminalu.

Malware investigation

Dio mog rada vezan je uz pregled sumnjivih datoteka, procesa i ponašanja na web okruženjima. Neki alati i skripte koje koristim za to nisu javno dostupni jer su vezani uz interni rad.

Traffic investigation

Bavim se i analizom prometa, najčešće kroz logove i obrasce ponašanja korisnika ili botova. Cilj je brže prepoznati neuobičajen promet i moguće uzroke opterećenja.

Web tehnologije

Kroz ovaj projekt korišteni su HTML za strukturu, CSS za dizajn i JavaScript za malu interakciju na kontakt formi. Stranica je napravljena kao jednostavno statičko web mjesto.
