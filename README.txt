NWKS završni projekt - Osobni portfolio

Ovaj projekt izrađen je kao završni projekt iz kolegija Napredna web korisnička sučelja.

Tema projekta je osobni portfolio. Odabrao sam ovu temu jer je jednostavna, ali opet dovoljno dobra za prikaz osnovne strukture web mjesta. Kroz portfolio sam ukratko prikazao osnovne informacije o sebi, posao kojim se bavim, vještine koje koristim i kontakt podatke.

Web mjesto se sastoji od naslovne stranice i tri podstranice:

1. index.html
   Početna stranica. Na njoj se nalazi kratak uvod, moja fotografija, osnovni opis i pregled glavnih područja kojima se bavim.

2. o-meni.html
   Podstranica s osnovnim informacijama o meni, poslu, interesima i hobijima. Ovdje sam dodao i objašnjenje zašto je stranica napravljena na jednostavan i pregledan način.

3. vjestine.html
   Podstranica na kojoj su opisane moje tehničke vještine, rad s Linux sustavima, Bash skriptama, analizom logova, web tehnologijama i sigurnosnim provjerama.

4. kontakt.html
   Kontakt stranica s osnovnim kontakt podacima, GitHub profilom i jednostavnom kontakt formom.

Korištene datoteke:

- index.html
- o-meni.html
- vjestine.html
- kontakt.html
- style.css
- script.js
- zlatko-slika.jpg

Korištene tehnologije:

HTML je korišten za strukturu stranica, odnosno za naslove, odlomke, navigaciju, slike, linkove, kartice i kontakt formu.

CSS je korišten za izgled stranice. Pomoću CSS-a određene su boje, fontovi, razmaci, raspored elemenata, izgled kartica, navigacije, gumba i prilagodba stranice manjim ekranima.

JavaScript je korišten samo za jednostavnu interakciju na kontakt formi. Kada korisnik ispuni formu i klikne gumb "Pošalji", prikazuje se poruka potvrde. Forma ne šalje stvaran e-mail, nego služi kao primjer osnovne interakcije na stranici.

Dizajn stranice

Stranica je napravljena jednostavno i pregledno. Koristi se sans-serif font jer je čitljiv na ekranu i dobro odgovara web stranicama. Korištene su plava, bijela i svijetlosiva boja. Plava boja daje ozbiljniji i profesionalniji dojam, dok svijetla pozadina olakšava čitanje sadržaja.

Navigacija se nalazi na vrhu stranice kako bi korisnik odmah mogao otvoriti sve glavne dijelove web mjesta. Sadržaj je podijeljen u odvojene sekcije i kartice kako stranica ne bi izgledala zbijeno ili nepregledno.

Pokretanje projekta

Za pokretanje projekta potrebno je sve datoteke držati u istoj mapi i otvoriti datoteku index.html u web pregledniku.

Važno je da se datoteka zlatko-slika.jpg nalazi u istoj mapi kao i HTML datoteke, jer se ta slika koristi na početnoj stranici i na stranici O meni.

Napomena

Projekt je izrađen kao statičko web mjesto. To znači da nema bazu podataka niti stvarno slanje poruka putem kontakt forme. Cilj projekta bio je prikazati osnovnu izradu web mjesta s više stranica, jednostavnim dizajnom, navigacijom i osnovnom JavaScript interakcijom.
